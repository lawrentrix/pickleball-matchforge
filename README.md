# Pickleball MatchForge

## Description
Pickleball MatchForge is a web-based product for pickleball player registration, matchmaking, regular game pairing, and tournament management.

## Current stack
- Next.js
- React
- TypeScript
- Tailwind CSS
- App Router

## Folder structure
- `frontend/` — Next.js web application
- `backend/` — future API/backend services
- `docs/` — architecture, tournament logic, and product documentation
- `scripts/` — automation/helper scripts

## Basic commands
```bash
cd frontend
npm install
npm run dev
npm run build
```
